About the game

Jet fighter the game was thought up with old arcade games in mind. The game, built with the Phaser Game Engine, is college project that incorperates the design and deployment of a software application.

The artwork for the jets comes from a member of gamedev.net who goes by the name of Prinz Eugn. To check out some of his artwork and for the sprite zip file used in this project follow the link.
